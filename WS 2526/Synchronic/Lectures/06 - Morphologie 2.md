# 06 - Morphologie 2: Wortbildung und Morphosyntax

## Mittel der Wortbildung

### Grundprinzip
Neue Wörter oder Wortformen können entweder:
1. Aus **eigenen sprachlichen Mitteln** gewonnen werden
2. Durch **Entlehnung** (Übernahme aus einer anderen Sprache)

### Verfahren mit eigenen sprachlichen Mitteln

1. **Flexion** (Beugung)
2. **Derivation** (Ableitung)
3. **Konversion** (Nullderivation)
4. **Komposition** (Zusammensetzung)
5. **Abbreviation** (Abkürzung)
6. **Amalgamierung** (Zusammenrückung)

## 1. Flexion

### Definition
Eine Wortform erfährt durch die Einbettung in einen Kontext **grammatikalische Anpassungen**.

### Veränderte Kategorien
- **Numerus** (Zahl: Singular | Dual | Plural)
- **Genus** (Geschlecht: Maskulinum | Femininum | Neutrum)
- **Tempus** (Zeit: Präsens | Präteritum | Futur...)
- **Modus** (Indikativ | Konjunktiv...)
- **Kasus** (Fall: Nominativ | Genitiv...)
- **Person** (1. | 2. | 3.)

### Wichtige Eigenschaft
Flexion **bringt keine neuen Wörter** hervor, sondern **neue Wortformen**.

### Arten der Flexion

#### a) Affixale Flexion
- Durch das Antreten grammatikalischer Morpheme
- **Beispiel**: Ich leg-e mein Amt nieder.

#### b) Innere Flexion
- Anpassungen innerhalb des Wortstamms
- **Beispiel**: Sie wär-en auch gerne dabei.

### Beispiele für Flexion

| Satz | Information | Kategorien |
|------|-------------|------------|
| Ich lege mein Amt-∅ nieder. | Akkusativ Singular Neutrum | Kasus, Numerus, Genus |
| Ich fühle mich in meinem Amt-∅ unwohl. | Dativ Singular Neutrum | Kasus, Numerus, Genus |
| Ich leg-e mein Amt nieder. | 1. Person Singular Indikativ Präsens | Numerus, Person, Modus, Tempus |
| Wir leg-t-en unsere Ämter nieder. | 1. Person Plural Indikativ Präteritum | Numerus, Person, Modus, Tempus |
| Sie wär-en auch gerne dabei. | 3. Person Plural Konjunktiv Präteritum | Numerus, Person, Modus, Tempus |

## 2. Derivation

### Definition
**Derivation** (Ableitung) ist ein Prozess, bei dem aus bestehenden Wörtern **neue Wörter** gebildet werden (ohne den Einsatz weiterer lexikalischer Morpheme).

### Eigenschaft
Die **Wortart kann sich ändern**.

### Beispiele aus dem Deutschen

| Ausgangswort | Abgeleitetes Wort | Prozess | Wortartänderung |
|--------------|-------------------|---------|-----------------|
| fassen | fass-bar | Verb → Adjektiv |
| grün | grün-lich | Adjektiv → Adjektiv |
| fassen | Fass-ung | Verb → Substantiv |
| Kärcher | kärcher-n | Substantiv → Verb |
| grün | er-grün-en | Adjektiv → Verb |
| Ding | Un-ding | Substantiv → Substantiv |

### Beispiele aus dem Neuštokavischen

| Ausgangswort | Abgeleitetes Wort | Prozess |
|--------------|-------------------|---------|
| rad 'Arbeit' | rad-nik 'Arbeiter' | Substantiv → Substantiv |
| pisati 'schreiben' | pis-a-ć 'schreiben' (Infinitiv) | Verb → Verb |

## 3. Konversion (Nullderivation)

### Definition
Bei der Konversion kommen **keine weiteren Morpheme** ins Spiel.

### Eigenschaft
Kann als **Sonderfall der Derivation** betrachtet werden.

### Beispiele

| Sprache | Ausgangswort | Abgeleitetes Wort | Wortartänderung |
|---------|--------------|-------------------|-----------------|
| Deutsch | rot (Adjektiv) | das Rot (Substantiv) | Adjektiv → Substantiv |
| Deutsch | leben (Verb) | das Leben (Substantiv) | Verb → Substantiv |
| Englisch | lecture (Substantiv) | to lecture (Verb) | Substantiv → Verb |

### Lexikalisierung

#### Definition
Wird aus einer Wortform (mit Flexionsmorphem) ein selbstständiges Wort erzeugt.

#### Beispiele

| Sprache | Ausgangswort | Lexikalisierte Form | Bedeutung |
|---------|--------------|---------------------|-----------|
| Russisch | ночь-ю (nóč'-ju) | ночью (noč'ju) | 'nachts' |
| | (Instr. Sg.) | | |
| Polnisch | czas-ami | czasami | 'manchmal' |
| | (Instr. Pl.) | | |
| Deutsch | tag-s | tags | 'am Tag' |
| | (Gen. Sg.) | | |

## 4. Komposition (Zusammensetzung)

### Definition
An der Wortbildung sind **zwei oder mehr lexikalische Morpheme** beteiligt.

### Beispiele
- **Deutsch**: Haus-tür
- **Russisch**: водопад (vod-o-pád) 'Wasserfall'

### Wichtige Eigenschaft
Komposita können **nicht nur auf Basis von Substantiven** gebildet werden.

#### Beispiele nach Wortarten

| Typ | Beispiel | Struktur |
|-----|----------|----------|
| **Substantiv + Substantiv** | Haustür | N + N |
| **Adjektiv + Substantiv** | Rotkohl | A + N |
| **Verb + Substantiv** | Waschmaschine | V + N |
| **Präposition + Substantiv** | Übergang | P + N |
| **Substantiv + Adjektiv** | kindlich | N + A |
| **Adjektiv + Adjektiv** | hellblau | A + A |

## 5. Abbreviation (Abkürzung)

### Definition
Am Wortkörper werden **lautliche Kürzungen** vorgenommen.

### Arten der Abbreviation

#### a) Teilkürzung
- **Beispiel**: russ. спецоперация → спец (spez)
- **Typ**: Teilkurzwort

#### b) Initialkürzung

##### Buchstabiertyp
- **Beispiele**: 
  - russ. РФ [ˌærˈæf] (Rossijskaja Federacija)
  - dt. LV [ˌelˈfau̯] (Landesverband)

##### Sprechtyp (Akronym)
- **Beispiele**:
  - russ. бомж [ˈbomʃ] (bez opredelënnogo mesta žitel'stva)
  - NATO, URZ

#### c) Silbenkürzung

##### Backclipping (Schwanzkürzung)
- **Beispiele**:
  - dt. Mofa < Motorfahrrad
  - dt. Vokuhila (vorne kurz, hinten lang)
  - engl. Auto(mobile) > Auto
  - dt. Uni(versität) > Uni

##### Frontclipping (Kopfkürzung)
- **Beispiele**:
  - Omnibus > Bus
  - Telefax > Fax

##### Mischtypen
- Kombination aus Front- und Backclipping möglich

## 6. Amalgamierung (Zusammenrückung)

### Definition
Fügungen aus Wörtern und/oder Wortformen **vereinen sich zu einem selbstständigen Wort**.

### Beispiele

| Sprache | Wort | Herkunft | Bedeutung |
|---------|------|----------|-----------|
| Deutsch | Dreikäsehoch | drei + Käse + hoch | kleines Kind |
| Deutsch | Stelldichein | stell' dich ein! | Treffen |
| Deutsch | nichtsdestoweniger | nichts desto weniger | trotzdem |
| Russisch | спасибо | спаси бог | 'rette Gott!' → 'danke' |
| Russisch | сумасшедший | с ума шедший | 'verrückt' |
| Russisch | вышеупомянутый | выше + упомянутый | 'der oben Erwähnte' |
| Neuštokavisch | takozvan | tako + zvan | 'sogenannt' |
| Englisch | goodbye | God be with you | 'Gott sei mit dir' |
| Latein/Deutsch | Noli me tangere | noli me tangere | 'berühre mich nicht' |
| Italienisch | tiramisù | tira mi su | 'zieh mich auf!' |

## Morpho-Phonologie

### Definition
Die **Morpho-Phonologie** (auch: Morphonologie oder Morphematik) ist die Analyseebene **zwischen Phonologie und Morphologie**.

### Untersuchungsgegenstand
- Lautliche Phänomene, die bei der Bildung von Wörtern und Wortformen wirken

### Beispiel aus dem Deutschen: Umlaut

| Wort | Suffix | Ergebnis | Wortart |
|------|--------|----------|---------|
| Not | -ig (bildet Adjektive) | nötig | Substantiv → Adjektiv |
| schwarz | -en (Infinitiv) | schwärzen | Adjektiv → Verb |
| braun | -e (Abstrakta) | Bräune | Adjektiv → Substantiv |
| Wand | -e (Plural) | Wände | Substantiv → Plural |

#### Erklärung
- Diese Phänomene gehen auf **alte Lautwandel** zurück
- Inzwischen zumeist **verschleiert**

### Beispiele aus den slavischen Sprachen: Konsonantenwechsel

#### Russisch

| Stamm | Wortform | Bedeutung |
|-------|----------|-----------|
| **друг** (drug) | 'Freund' | Nom. Sg. |
| **друзья** (druz'ja) | 'Freunde' | Nom. Pl. |
| **дружба** (družba) | 'Freundschaft' | Substantiv |

**Wechsel**: g → ž / z' (je nach Position)

#### Neuštokavisch

| Stamm | Wortform | Bedeutung |
|-------|----------|-----------|
| **rad-nik** | 'Arbeiter' | Nom. Sg. |
| **rad-nič-e** | 'Arbeiter!' | Vokativ Sg. |
| **rad-nic-i** | 'Arbeiter' | Nom. Pl. |

**Wechsel**: k → č / c (je nach Position)

#### Polnisch

| Stamm | Wortform | Bedeutung |
|-------|----------|-----------|
| **pis-a-ć** | 'schreiben' | Infinitiv |
| **pisz-ę** | 'ich schreibe' | 1. Sg. Präsens |

**Wechsel**: s → sz (je nach Position)

### Weitere morpho-phonologische Prozesse

#### 1. Ablaut (Vokalwechsel im Wortinneren)

| Sprache | Beispiel | Bedeutung |
|---------|----------|-----------|
| **Deutsch** | binden - Band - Bund | 'binden' - 'Band' - 'Bund' |
| **Kroatisch** | reći - riječ - rok | 'sagen' - 'Wort' - 'Frist' |

#### 2. Vokalharmonie
- Vokalwechsel in Affixen
- Gesteuert durch Vokalismus im Basismorphem

| Sprache | Beispiel | Bedeutung |
|---------|----------|-----------|
| **Finnisch** | talo-ssa | 'im Haus' |
| | Helsingi-ssä | 'in Helsinki' |

## Allomorphe (Morphemvarianten)

### Definition
Morpheme können verschiedene **lautliche Gestalten** haben und trotzdem dieselbe Funktion erfüllen.

### Allomorphie
- Die Erscheinung verschiedener lautlicher Gestalten
- Die betreffenden Morpheme heißen **Allomorphe** oder **Morphemalternanten**

### Determinierung
Die Wahl des Allomorphs ist durch den **Kontext** bestimmt (vgl. Allophone).

### Arten der Allomorphie

#### 1. Morpho-phonologisch verursacht

##### Ursache
Lautliche Umgebung

##### Basis
Gemeinsame (abstrakte, virtuelle) Form

##### Beispiele (aus vorherigen Folien)
- Russisch: °drug → drug / druž / druz'
- Neuštokavisch: °nik → nik / nič

#### 2. Lexikalisch bedingt

##### Ursache
Vom lexikalischen Morphem bestimmt, an das das Morphem antritt

##### Beispiel Englisch: Pluralbildung

| Form | Plural |
|------|--------|
| dog | dog-s |
| cat | cat-s |
| ox | ox-en |
| child | child-ren |

##### Beispiel Deutsch: Pluralbildung

| Singular | Plural |
|----------|--------|
| Fisch | Fisch-e |
| Finger | Finger-∅ |
| Stimme | Stimm-en |
| Kind | Kind-er |

## Morphosyntax

### Definition
Morphosyntax ist ein Untersuchungsfeld, das sich mit den **Wechselwirkungen der Morphologie und der Syntax** beschäftigt.

### Untersuchungsgegenstand
Wie Wörter angepasst ("flektiert", "gebeugt", "abgewandelt") werden, damit sie innerhalb eines Satzes grammatikalische Funktionen ausüben.

### Grammatische Kategorien

#### 1. Numerus (Zahl)
- **Beschreibt**: Mengenwert von bezeichneten Entitäten
- **Ausprägungen**: 
  - Singular (Einzahl)
  - Dual (Zweizahl) - nur in einigen Sprachen
  - Plural (Mehrzahl)
- **Beispiel Slowenisch**:
  - Ona gre 'sie geht' [Sg.]
  - Onedve greta 'sie beide (f.) gehen' [Du.]
  - One gredo 'sie (f.) gehen' [Pl.]

#### 2. Person
- **Beschreibt**: Rolle der bezeichnenden Entitäten im Sprechakt
- **Ausprägungen**:
  - 1. Person (sprechende): ich, wir
  - 2. Person (angesprochene): du, ihr
  - 3. Person (besprochene): er/sie/es, sie

#### 3. Genus (Geschlecht)
- **Beschreibt**: Klassifiziert Substantive unter Rückgriff auf außersprachliche Kategorien
- **Ausprägungen** (dt., slav.): Maskulinum, Femininum, Neutrum
- **Wichtig**: Zuordnung unterscheidet sich von Sprache zu Sprache

##### Beispiele
| Deutsch | Polnisch | Neuštokavisch | Italienisch |
|---------|----------|---------------|-------------|
| Sonne (f.) | słońce (n.) | sunce (n.) | sole (m.) |
| Apfel (m.) | jabłko (n.) | jabuka (f.) | mela (f.) |

#### 4. Modus
- **Beschreibt**: Klassifiziert die Satzaussage nach ihrem Bezug zur Realität
- **Ausprägungen** (Auswahl):
  - Indikativ (Wirklichkeitsform)
  - Konjunktiv (Möglichkeitsform)
  - Optativ (Wunschform)
- **Beispiel**:
  - Dörte ist heute so nervig (Indikativ)
  - Wäre Dörte heute nicht so nervig, ... (Konjunktiv)

#### 5. Tempus (Zeit)
- **Beschreibt**: Lokalisiert die Satzaussage in der zeitlichen Dimension
- **Ausprägungen** (Auswahl):
  - Präsens (Gegenwart)
  - Futur (Zukunft)
  - Präteritum (Mitvergangenheit)
  - Perfekt (Vergangenheit)

#### 6. Kasus (Fall)
- **Beschreibt**: Klassifiziert eine Wortform nach ihrer Abhängigkeit von anderen Wörtern
- **Ausprägungen** (Auswahl):
  - Nominativ (nur vom Verb abhängig)
  - Genitiv (Zugehörigkeit ausgedrückt)
  - Dativ
  - Akkusativ
- **Wichtig**: Zahl und Auswahl variiert von Sprache zu Sprache

#### 7. Aspekt
- **Nur in den slavischen Sprachen** (nicht im Deutschen)
- **Beschreibt**: Klassifiziert die Satzaussage nach ihrer Erstreckung in der Zeit oder Abgeschlossenheit
- **Ausprägungen** (slav.):
  - Vollendet (russ. совершенный вид)
  - Unvollendet (russ. несовершенный вид)

### Verteilung der grammatischen Kategorien auf Wortarten

| Wortart | Kategorien |
|---------|------------|
| **Substantive** | Kasus, Numerus, Genus |
| **Adjektive** | Kasus, Numerus, Genus |
| **Pronomina** | Kasus, Numerus, Genus, Person |
| **Verben** | Numerus, Person, Tempus, Modus, Aspekt |
