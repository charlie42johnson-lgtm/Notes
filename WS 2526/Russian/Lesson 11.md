### Verb: able to / can: мочь + infinitive

| Сейчас     |        |
| ---------- | ------ |
| Я          | могу   |
| Ты         | можешь |
| Он Она Оно | может  |
| Мы         | можем  |
| Вы         | можете |
| Они        | могут  |
| **Вчера**  |        |
| Он         | мог    |
| Она        | могла  |
| Они        | могли  |

### Verb: be able to / learned (unfinished): уметь + infinitive НСВ

| Сейчас     |        |
| ---------- | ------ |
| Я          | умею   |
| Ты         | умеешь |
| Он Она Оно | умеет  |

| Они        | умеют  |
| **Вчера**  |        |

### Verb: must: должен + infinitive

| Сейчас      |               |
| ----------- | ------------- |
| Я, Ты, Он   | должен        |
| Я, Ты, Она  | должна        |
| Мы, Вы, Они | должны        |
| **Вчера**   |               |
| Он          | должен + был  |
| Она         | должна + была |
| Они         | должны + были |
